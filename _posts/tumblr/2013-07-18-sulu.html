I’m diving head first into Ruby by writing my own gem : Sulu. This project will be an adaptation of my previously released website with the same name. Sulu was originally built to help me find tabs while I was playing guitar. Since it’s connected to Last.fm, I can start pretty much any song in my music library and Sulu will find the best tab according to what’s currently playing. My plan is to write the gem with the same functionalities and then rebuild the website around it.
